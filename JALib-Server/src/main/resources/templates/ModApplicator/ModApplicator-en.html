Mod Automatic Application Guide

To automatically apply the mod, you must install version 'JALib 1.0.0 beta 10' or higher and run it once.

If you want to install the mod manually, you can click the manual installation button.

If the automatic installer does not open, you can try again by clicking the rerun button.

If automatic installation continues to fail, please run version 'JALib 1.0.0 beta 10' or higher again and try.
